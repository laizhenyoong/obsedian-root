
What is JSON?
- JSON is a lightweight data exchange format.
- JSON is easy for humans to read and write.
- JSON is built on a collection of key/value pairs and an ordered list of values.

```json
{ 
	"name": "Zhen Yoong", 
	"age": 24, 
	"isEmployed": true, 
	"hobbies": ["football", "coding"]
}
```
