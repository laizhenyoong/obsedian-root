
What is MongoDB?
- MongoDB is a [[NoSQL]] database that stores data in flexible, [[JSON]]-like documents. 
- MongoDB provides high availability, horizontal scaling, and geographic distribution of data.


```json
{
	title: "Post Title 1",
	body: "Body of post.",
	category: "News",
	likes: 1,
	tags: ["news", "events"],
	date: Date()
}
```
